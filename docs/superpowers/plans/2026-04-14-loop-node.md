# LoopNode 循环节点实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox syntax for tracking.

**Goal:** 为 orchestrator 编排引擎引入 LoopNode 循环复合节点，替代静态展开的审核-修订循环

**Architecture:** LoopNode 作为特殊复合节点嵌入主 DAG，内部封装可重复执行的子 DAG。主 DAG 保持无环，循环语义封装在 LoopNode 内部。支持嵌套循环、按轮次保留历史、可配置耗尽策略。

**Tech Stack:** Go 1.22, errgroup, text/template
