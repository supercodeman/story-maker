🔄 数据流全景 — 用户点击「生成章节」
Step 1
用户点击「生成章节」→ handleFullChapter()
Step 2
前端调用 workflowStore.submitWorkflow()
Step 3
HTTP POST → WorkflowHandler.Submit()
Step 4
WorkflowService → BuildFullChapterGraph() 构建 DAG
Step 5
DAG 引擎执行: outline → draft (tools) → review_loop
Step 6
Provider.GenerateText() 调用 AI，含 fallback 降级
Step 7
WebSocket 推送进度 → 前端实时更新 UI
Step 8
用户确认接受 → 版本保存至 MySQL

  • 世界观 (WorldView)
  • 伏笔线索 (Foreshadow)

  • 章节内容向量化

  • 动态上下文窗口
Phase 3: 递归摘要树
Chapter Summary Tree
  • 章节级摘要
  • 卷级聚合摘要
  • 全书递归压缩
  • 上下文预算分配
