🔧 Harness Engineering — 编排基础设施全景

核心价值

把「调哪个模型、怎么调、失败了怎么办」的复杂度封装在基础设施层。业务代码只需声明 taskType + prompt 即可获得完整的执行保障：能力路由、三级降级、Token 预算、结果持久化、实时通知。
